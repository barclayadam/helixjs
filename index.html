Helixjs

HelixJS is a JavaScript library that provides a solid foundation for creating rich web applications

Download this project as a .zip file Download this project as a tar.gz file

HelixJS is a JavaScript library that provides a solid foundation for creating rich web applications, building on the Knockout library.

License: MIT http://www.opensource.org/licenses/mit-license.php

Supported (Tested) Browsers:

  • IE8+ (Yes, it's time to drop support for IE7 and less)
  • Chrome (Latest)
  • Firefox (Latest)

Other browsers that match the feature sets of Chrome and Firefox are likely to work as well as the other browsers, they are just not yet fully tested and supported.
